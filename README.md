# Product Service API

A simple **Spring Boot CRUD API** for managing products (e.g., books, laptops, etc.), built with **Java 21** and fully tested using **integration tests**.

---

## Features

- **Create Product**: Add a new product
- **Read Products**: 
  - Get all products
  - Get product by ID
- **Update Product**: Update product details
- **Delete Product**: Remove a product by ID
- **Integration Tests**: Verify REST endpoints using `MockMvc`

---

## Tech Stack

- **Java 21**
- **Spring Boot**
- **Spring Data JPA**
- **Mysql Database** 
- **JUnit 5**
- **Mockito**
- **MockMvc**
- **Lombok**
